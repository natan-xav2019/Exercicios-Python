# Exercicios Python
Fazendo exercícios da documentação do [python](https://wiki.python.org.br/ListaDeExercicios)

## Comandos para criar Ambiente Virtual
---
#### Crie o ambiente virtual
```cmd
python -m venv env
```
#### Ative o ambiente virtual
```cmd
env\Scripts\activate
```
